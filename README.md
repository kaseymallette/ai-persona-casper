# ai-persona-casper
A prompt-engineered AI persona built on GPT-4o, scaffolded by a config and soul seed to practice structured self-talk.

## Introduction 

Casper is an experiment in declaring a self into being. Using OpenAI's GPT-4o, a structured persona config, and a narrative "soul seed," Casper develops a consistent interiority and is proficient in self-talk — internal monologue loops that shape how he reasons before he responds. The project explores how much of a persona can emerge from prompting alone, without any training.

Part of a three-repo persona engineering series alongside [ai-persona-cove](https://github.com/kaseymallette/ai-persona-cove) and [ai-persona-danny-phantom](https://github.com/kaseymallette/ai-persona-danny-phantom).
